# Foreign Tourists for customized regional tourism

주제: 외국인 관광객 관심/활동 빅데이터 기반 맞춤형 지역관광 전략 제안

아이디어 제안 배경 및 필요성:  
최근 외국인 관광객 수 급증  → 하지만 서울 중심 집중 현상으로 지방 관광지 소비율은 아직 여전히 낮음. 따라서 외국인 관광객의 실제 소비, 이동, 문화활동 데이터를 분석해 지역 맞춤형 관광정책을 제시할 필요성이 있음.

관광 정책에서“찾아가는 문화”, “지역 브랜드 관광”, “분산형 관광지 육성” 기조가 강화된 것을 볼 수 있음. 예) 문화체육관광부의 지역 문화 예술인, 창작자 지원 확대
https://knto.or.kr/pressRelease/549005?

-> 인구 감소, 지방 소멸 이슈가 떠오르고 있는 가운데 지역 문화관광 활성화가 정부의 중요한 과제로 떠오름
